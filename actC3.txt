##################################### Bloque 2 ############################################

Fila del Cajero:
Cola común (FIFO), se atiende por orden de llegada.

Impresora:
Orden de salida = mismo orden de llegada (FIFO).

Guardia (prioridades ejemplo):

Raspón -> 1
Fiebre alta -> 50
Infarto -> 100 

Examen Final:
Sale primero el promocionado (mayor prioridad).

Buffet:
enqueue() para agregar al final (evita colarse).

Scheduler (SO):

Sistema -> 50
Usuario -> 10
Si se sobrecarga: se ejecutan primero los de mayor prioridad, los demás esperan (starvation posible).

Embarque:
Si misma prioridad -> FIFO entre ellos.

Tickets IT:

Servidor caído -> 999
Fondo pantalla -> 0

Ejemplo código cola prioridad:

class PriorityQueue {
  constructor() {
    this.items = [];
  }

  enqueue(element, priority) {
    this.items.push({ element, priority });
    this.items.sort((a, b) => b.priority - a.priority);
  }

  dequeue() {
    return this.items.shift();
  }
}class Tarea {
    String nombre;
    int prioridad;

    public Tarea(String nombre, int prioridad) {
        this.nombre = nombre;
        this.prioridad = prioridad;
    }
}

public class Main {
    public static void main(String[] args) {
        PriorityQueue<Tarea> cola = new PriorityQueue<>(
            (a, b) -> b.prioridad - a.prioridad // mayor prioridad primero
        );

        cola.add(new Tarea("Spotify", 10));
        cola.add(new Tarea("Sistema", 50));
        cola.add(new Tarea("Fondo pantalla", 0));

        while (!cola.isEmpty()) {
            Tarea t = cola.poll();
            System.out.println(t.nombre + " - " + t.prioridad);
        }
    }
}
